# Data Provenance

Redacted manifests and dataset metadata in this repository were derived from local pilot samples built during the AIRA corpus study workflow.

Included provenance fields:
- `repo_name`
- `repo_url`
- `pr_identifier`
- `language`
- `path`
- `source_license`
- `attribution_strength`
- `line_count`
- `content_sha256`

The public artifact intentionally omits raw source content while preserving enough structure to:
- audit sampling composition
- inspect repo/language balance
- verify sample identity against local reconstructions
